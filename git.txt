git status -s 
me muestra los modificaciones.
git add . 
agrega todos los archivos de la carpeta al siguiente commit
git commit -m "mensaje del commit"
guarda el commit en github
git push 
guarda commit en el servidor de github
git log
muestra los commits que existen en el repositorio
git reset --hard head~3
borra los ultimos 3 commits y vuelve para atras
